# 30-2 Implementar login seguro e sessão server-side

[Voltar para lista de tarefas](./tasks.md)

## Description
Implementar fluxo de login/logout com sessão segura validada no servidor, utilizando Supabase Auth e cookies protegidos para impedir armazenamento de tokens sensíveis no client.
