# 34-7 Hardening CI Playwright para PR sem secrets

## Description
Ajustar o pipeline de CI para rodar smoke tests de Playwright em PR sem quebrar por ausência de secrets, e executar a suíte auth-real apenas quando o ambiente Supabase estiver disponível.

## Requirements
- [x] Workflow de Playwright com gatilho em `pull_request`.
- [x] Smoke tests executam sem depender de secrets de Supabase.
- [x] Auth-real roda somente quando env obrigatória está disponível.
- [x] Pipeline não falha por falta de env em PR de contexto restrito.

## Implementation Plan
1. Adicionar trigger `pull_request` no workflow de Playwright.
2. Incluir etapa de verificação de env e output `has_supabase_env`.
3. Separar execução:
   - smoke sempre;
   - auth-real e build apenas com env.
4. Marcar testes de autenticação real com tag dedicada.
5. Validar type-check e lint após ajustes.
