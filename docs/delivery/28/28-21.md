Tornar os testes de IA no Playwright determinísticos por meio de mocks dos endpoints de IA, removendo dependência de quota, latência e indisponibilidade de provedores externos.

## Requirements
- [x] Mockar respostas de sucesso e erro dos endpoints `/api/ai/*` usados nos cenários E2E.
- [x] Cobrir cenários de fallback e de limite de taxa sem chamar API externa real.
- [x] Garantir que asserts validem UX (mensagens, loading, estados), não custo/quota.

## Implementation Plan

- Padronizar interceptações com `page.route` e payloads fixos.
- Separar cenários "mockados" de qualquer smoke opcional externo.
- Atualizar documentação de execução para deixar claro o modo determinístico.
