# 27-9 Desenvolver recomendações de orçamento

[Voltar para lista de tarefas](./tasks.md)

## Description
Implementar camada de recomendação de orçamento orientada por previsão para apoiar decisões de alocação entre campanhas/adsets.

## Requirements
- [x] Definir estratégia de recomendação com base nas previsões.
- [x] Gerar sugestão de aumentar/reduzir/manter orçamento por entidade.
- [x] Exibir impacto esperado (trade-off risco x resultado).
- [x] Registrar racional da recomendação para auditoria.

1. Definir regras e função objetivo da recomendação.
2. Implementar cálculo por entidade com restrições mínimas.
3. Produzir payload de recomendação com justificativa.
4. Validar consistência em cenários distintos.
