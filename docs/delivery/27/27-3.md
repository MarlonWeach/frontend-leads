# 27-3 Implementar análise de sazonalidade

[Voltar para lista de tarefas](./tasks.md)

## Description
Desenvolver a camada de análise de sazonalidade para identificar padrões recorrentes (semanal/mensal) nas séries de performance e disponibilizar esses sinais para melhorar as previsões iniciadas na 27-2.

## Requirements
- [x] Detectar sazonalidade semanal e mensal nas séries por adset/campanha.
- [x] Gerar features sazonais reutilizáveis pelo pipeline da 27-2.
- [x] Documentar critérios de ativação/desativação de sazonalidade por série.
- [x] Garantir execução idempotente e custo computacional controlado.

## Implementation Plan
1. Definir método de detecção (decomposição/estatística) alinhado ao baseline.
2. Implementar geração de sinais sazonais em pipeline de dados.
3. Integrar features ao fluxo de treino/inferência.
4. Validar impacto e registrar limitações.

## Verification
- [x] Sinais sazonais são gerados para séries elegíveis.
- [x] Pipeline roda sem regressão no tempo de execução esperado.
- [x] Evidência de resultado documentada.
