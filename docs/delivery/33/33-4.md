## Implementation Summary
- Endpoint atualizado: `app/api/adset-goals/dashboard/route.ts`
  - `delivered_reference`:
    - usa `goal.volume_captured` quando informado;
    - fallback para soma de leads no período (`adset_insights`).
  - `volume_remaining`:
    - `max(0, volume_contracted - delivered_reference)`.
  - `leads_needed_daily`:
    - calculado a partir do `volume_remaining` e `days_remaining`.
  - Competência mensal:
    - `goalStart = contract_start_date`;
    - `competenceEnd = endOfMonth(goalStart)`.
  - status:
    - `atingido` quando referência >= contratado;
    - caso contrário, baseado na razão entre entrega de referência e entrega esperada até a data.

- Tipagem atualizada:
  - `src/types/adsetGoalsDashboard.ts`:
    - adicionados `delivered_reference` e `volume_remaining`.

- UI ajustada:
  - `src/components/metas/AdsetGoalCard.tsx`
    - progresso mostra `leads referência`;
    - bloco de leads/dia mostra `volume restante`.
