## Fórmulas Implementadas
- ROI:
  - `ROI (%) = ((receita_gerada - investimento) / investimento) * 100`
- Margem:
  - `Margem (%) = ((revenue - custo) / revenue) * 100`

## Implementation Summary
- Backend (`app/api/adset-goals/dashboard/route.ts`):
  - adicionados no payload de métricas:
    - `revenue_generated`
    - `cost_investment`
    - `roi_percentage`
    - `margin_percentage`
  - regra de receita:
    1. usa `goal.revenue_generated` quando presente;
    2. fallback para `goal.average_ticket * delivered_reference` quando disponível;
    3. caso contrário, mantém `null`.
  - investimento usa `total_spend` agregado.

- Tipagem (`src/types/adsetGoalsDashboard.ts`):
  - incluídos novos campos de métricas comerciais.

- UI (`src/components/metas/AdsetGoalCard.tsx`):
  - exibição de `ROI` e `Margem` no bloco de métricas;
  - função de formatação percentual para exibição consistente.
