# 33-10 Filtros por cards de resumo em `/metas`

[Voltar para lista de tarefas](./tasks.md)

## Description
Tornar os cards de resumo no topo de `/dashboard/metas` interativos para funcionar como filtro rápido de status dos adsets exibidos na grade.

## Requirements
- [x] Cards do resumo (`Total`, `No Prazo`, `Atenção`, `Atrasado`, `Crítico`, `Atingido`) clicáveis.
- [x] Clique em card de status aplica filtro para exibir apenas itens correspondentes.
- [x] Clique em `Total` limpa o filtro de status e volta a exibir todos.
- [x] Estado ativo visual no card selecionado.
- [x] Clique no mesmo card ativo faz toggle para limpar o filtro.

## Implementation Plan
1. Mapear os cards de resumo para botões com estado ativo.
2. Reutilizar o estado de filtros já existente (`filters.status`) para aplicar seleção única via resumo.
3. Adicionar função de toggle (selecionar/limpar) por card.
4. Manter compatibilidade com painel de filtros avançados.
